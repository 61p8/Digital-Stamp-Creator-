## แอปที่มีในชุดนี้

### App 1 · ตราประทับรูปร่างต่างๆ (`Sign_Stamp_Creater.html`)

สร้างตราประทับดิจิทัลได้ 8 รูปร่าง พร้อมตั้งค่าข้อความ 3 บรรทัดอิสระ

- รูปร่าง 8 แบบ: วงกลม, วงกลมเส้นเดียว, สี่เหลี่ยม, สี่เหลี่ยมเส้นเดียว, สี่เหลี่ยมมุมมน (2 แบบ), วงรี, ไม่มีขอบ
- แต่ละบรรทัด (บน/กลาง/ล่าง) เลือกฟอนต์, ขนาด, สไตล์ B/I/U อิสระ
- โหลดฟอนต์จากเครื่องได้ (Chrome/Edge)
- เลือกสี: 12 preset + color picker + hex code
- ลากเส้นแบ่งใน preview เพื่อปรับสัดส่วน · ลากข้อความขึ้น-ลงเพื่อจัดตำแหน่ง
- บันทึก PNG / คัดลอกรูป / พิมพ์ / บันทึกค่า .json
- **บันทึกอัตโนมัติ** ใน browser — เปิดครั้งหน้าจำค่าได้

### App 2 · ตรายางสี่เหลี่ยม (`Other_Stamp_Creater.html`)

- **บันทึกอัตโนมัติ** ใน browser

## วิธีใช้

### แบบที่ 1: เปิดในเครื่อง

ดับเบิลคลิก `index.html` → เลือกแอปที่ต้องการ → ใช้ได้ทันที

### แบบที่ 2: Host บน GitHub Pages (แชร์ในทีม)

1. สร้าง repo ใหม่บน GitHub (Public)
2. Upload ไฟล์ทั้งหมด (`index.html`, `Sign_Stamp_Creater.html`, `Other_Stamp_Creater.html`) เข้า repo
3. ไปที่ **Settings** → **Pages**
4. เลือก **Source: Deploy from a branch**
5. เลือก **Branch: main** / **Folder: / (root)** → กด Save
6. รอ 1-2 นาที จะได้ URL `https://[ชื่อผู้ใช้].github.io/[ชื่อ-repo]/`
7. ส่ง URL ให้คนในทีมเปิดหน้าหลักได้เลย

## เรื่องการบันทึกค่า (สำคัญ)

ข้อมูลทั้งหมดเก็บอยู่ในเครื่องผู้ใช้ **ไม่มีการส่งข้อมูลไปไหน**

- **บันทึกอัตโนมัติ**: ทุกการเปลี่ยนแปลงเซฟลง browser localStorage ทันที
- **แยกกันสมบูรณ์**: คุณ A กับคุณ B เปิด URL เดียวกัน — เห็นค่าของตัวเองคนละหน้า ไม่ปนกัน
- **อยู่กับ browser**: เปลี่ยนเครื่อง / เปลี่ยน browser = เริ่มใหม่
- **ปุ่ม "ล้าง"**: ลบค่าที่บันทึกไว้ทั้งหมด กลับเป็นค่าเริ่มต้น
- **ปุ่ม "บันทึกค่า"**: download ไฟล์ .json ไว้ backup หรือแชร์ template
