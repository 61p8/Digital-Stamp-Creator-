# เครื่องมือสร้างตราประทับ · Digital Stamp Creator

แอปสร้างตราประทับดิจิทัล (印鑑) เปิดผ่านเบราว์เซอร์ ไม่ต้องติดตั้ง

**Created by SITTISAK PE DNTH** · v1.7

---

## ฟีเจอร์

- รูปร่าง 8 แบบ: วงกลม, วงกลมเส้นเดียว, สี่เหลี่ยม, สี่เหลี่ยมเส้นเดียว, สี่เหลี่ยมมุมมน (2 แบบ), วงรี, ไม่มีขอบ
- แต่ละบรรทัด (บน/กลาง/ล่าง) เลือกฟอนต์, ขนาด, สไตล์ B/I/U อิสระ
- โหลดฟอนต์จากเครื่องได้ (Chrome/Edge) — ใช้ฟอนต์ที่ติดตั้งใน Windows ได้ทุกตัว
- เลือกสี: 12 preset + color picker + hex code
- ลากเส้นแบ่งใน preview เพื่อปรับสัดส่วน 3 ส่วน
- ลากข้อความขึ้น-ลงเพื่อจัดตำแหน่ง
- บันทึก PNG / คัดลอกรูป / พิมพ์
- บันทึก/โหลดไฟล์ตั้งค่า .json
- **บันทึกอัตโนมัติในเครื่อง** — เปิดครั้งหน้าจำค่าได้

## โครงสร้างหน้าแอป

- `index.html` = หน้าหลักรวมฟีเจอร์ (Feature Hub)
- `sign.html` = ฟีเจอร์ Sign Stamp (เดิมจากหน้าแอปหลัก)
- `approve.html` = ฟีเจอร์ Approve Stamp พร้อมช่องเซ็นแยก 3 ช่อง (Approved / Checked / Writer)

## วิธีใช้

### แบบที่ 1: เปิดในเครื่อง

ดับเบิลคลิก `index.html` → ใช้ได้ทันที

### แบบที่ 2: Host บน GitHub Pages (แชร์ในทีม)

1. สร้าง repo ใหม่บน GitHub (Public)
2. Upload ไฟล์ `index.html` เข้า repo
3. ไปที่ **Settings** → **Pages**
4. เลือก **Source: Deploy from a branch**
5. เลือก **Branch: main** / **Folder: / (root)** → กด Save
6. รอ 1-2 นาที จะได้ URL `https://[ชื่อผู้ใช้].github.io/[ชื่อ-repo]/`
7. ส่ง URL ให้คนในทีมเปิดได้เลย

## เรื่องการบันทึกค่า (สำคัญ)

ข้อมูลทั้งหมดเก็บอยู่ในเครื่องผู้ใช้ **ไม่มีการส่งข้อมูลไปไหน**

- **บันทึกอัตโนมัติ**: ทุกการเปลี่ยนแปลงเซฟลง browser localStorage ทันที
- **แยกกันสมบูรณ์**: คุณ A กับคุณ B เปิด URL เดียวกัน — เห็นค่าของตัวเองคนละหน้า ไม่ปนกัน
- **อยู่กับ browser**: เปลี่ยนเครื่อง / เปลี่ยน browser = เริ่มใหม่
- **ปุ่ม "ล้าง"**: ลบค่าที่บันทึกไว้ทั้งหมด กลับเป็นค่าเริ่มต้น
- **ปุ่ม "บันทึกค่า"**: download ไฟล์ .json ไว้ backup หรือแชร์ template

## ความปลอดภัย

ไฟล์นี้เป็น HTML + CSS + JavaScript ล้วนๆ:

- ไม่มีการเชื่อมต่อ server (ยกเว้น Google Fonts สำหรับโหลดฟอนต์)
- ไม่ track / ไม่ส่งข้อมูลผู้ใช้
- เปิด `index.html` ด้วย Notepad อ่าน source code ได้ทุกบรรทัด
- ตรวจกับ VirusTotal ได้

## Workflow Approve (3 Sign)

1. เข้า `approve.html`
2. คลิก `ไปทำ Sign` ของช่อง Approved / Checked / Writer
3. ทำตราใน `sign.html` แล้วคลิก `เสร็จ: ส่งไป Approve`
4. กลับมาที่ `approve.html` และลากตำแหน่งลายเซ็นได้
